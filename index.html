Projects

LANE FINDING

The project aimed to detect lanes in video footage taken using a camera attached to the front of the car. Used OpenCV library in Python to achieve the task. Image perspective warping was done to get a bird's-eye view of the lanes. Different color spaces like HSL, HSV, YCbCr, etc., were tested to extract lanes lines under different lighting conditions. Finally, Curve fitting was used to estimate the part of the lane that was not extracted properly.

FACIAL KEYPOINT DETECTION AND AUGMENTATION

A CNN model was trained in Pytorch to detect 68 key points using Youtube Faces DB Dataset. Pre-processing included resizing the image into a fixed size and normalizing the pixel value between 0 to 1. To reduce overfitting of the model, images were randomly shifted horizontally and vertically. Further, these key points were used to augment objects like goggles on the face.

TRYIT‑POSENET

It was an idea to let people buy clothes after trying them virtually during online shopping with a basic prototype created using ml5.js. Facial Attendance System

This project was created during the Smart Gujarat Hackathon 2020. The project aimed to remove the manual effort made by a teacher to mark the attendance of the class. A camera would be mounted in the class to click multiple images at different time intervals. A pre-trained MTCNN was used to detect faces from the images, and FaceNet was used to generate the embeddings of the faces. These embeddings were matched with embeddings present in the database to identify a person.
